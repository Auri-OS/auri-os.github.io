About AuriOS

Freedom First

AuriOS is built on the foundation of freedom. Fully open-source, it empowers users to explore, modify, and understand their system. Your computer should belong to you — not to the software that runs it.

Privacy by Principle

AuriOS respects your data and your digital space. No hidden telemetry, no silent data collection — transparency is at the core of its design. Your privacy isn’t an option; it’s a promise.

Minimal by Design

AuriOS embraces simplicity and efficiency. Every component is carefully chosen, every feature justified. The result: a clean, lightweight system that does only what’s necessary — and does it well.

Continuous Learning

This project is a learning adventure in operating system development, from low-level programming to system architecture.

Technical Features

Minimal Kernel

An optimized kernel for performance and simplicity, written primarily in C and Assembly.

Memory Management

Efficient memory management system with dynamic allocation and process protection.

Shell Interface

Interactive shell with support for basic commands and custom scripts.

File System

Implementation of a simple and efficient file system for data management.

Modular

Modular architecture allowing easy addition of new features.

Security

Built-in security mechanisms to protect the system and user data.
